# Project Overview
**This project is a console-based shopping cart application written in C.**
**It simulates a simple store where users can add/remove items, calculate totals, apply discounts, and print a receipt.**
**The project follows Test-Driven Development (TDD) principles to ensure all core functionality is thoroughly tested before the UI is built.**
